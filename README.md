## 👨🏻‍💻 Running Locally
Recommended extensions for VSCode:

- Tailwind CSS IntelliSense.
- Astro.

Clone or fork the repository:
```bash
git@github.com:Brian-Kariu/blog.git
```

Install dependencies:
```bash
pnpm install
Run the development server:
pnpm dev
```
